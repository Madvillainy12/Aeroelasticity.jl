Performing a Stability Analysis

The following method may be used to perform a stability analysis for a model.

AerostructuralDynamics.get_eigen — Function
get_eigen(model::TM, x, p; kwargs...)

Return the eigenvalues, left eigenvector matrix, and right eigenvector matrix corresponding to the model for state variables x and parameters p.

For in-place models, the number of eigenvalues to compute may be specified using the nev keyword argument.

source
